# Zynq-7100 (Xilinx SoC)


**核心定义**：
Zynq-7000 系列中的顶级全可编程片上系统 (All Programmable SoC)。它在一块硅片上融合了硬件逻辑 (PL 端) 与软件处理器 (PS 端)。

**内部异构架构分工**：
* **PL 端 (Programmable Logic)**：等效于 Kintex-7 系列的纯硬件 FPGA 架构。拥有极强的并行 DSP 算力，适合做海量数据的流式处理（如 `[[LVDS解串]]`、超声并发多通道 `[[DAS (延迟求和)]]`、`[[硬件检波]]`）。
* **PS 端 (Processing System)**：双核 ARM Cortex-A9 处理器。适合跑 Linux/裸机 C 代码，负责千兆网/TCP协议栈、系统调度、外设配置（如通过 SPI 配置 ADC 寄存器）。

**系统瓶颈与架构决策 (超声应用)**：
* **BRAM 存储墙**：Zynq-7100 片内的 Block RAM (BRAM) 极限约为 26.5 Mb。对于传统的流式 DAS 处理绰绰有余，但面对需要存储全矩阵捕获 (FMC) 数据的 `[[TFM (全聚焦方法)]]` 时，BRAM 会被瞬间击穿。
* **架构折中**：在高端 NDT 仪器中，通常用 Zynq 的 PL 端完成前端高带宽压缩（DAS），而将 TFM 矩阵计算上移至拥有海量内存的上位机 PC 或 GPU (CUDA) 处理。
