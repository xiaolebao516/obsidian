# 模数转换器 (ADC)

#Hardware #DSP #数字信号处理

**核心定义**：
模拟-数字转换器 (Analog-to-Digital Converter)。它是跨越模拟世界（连续电压）与数字世界（离散 0 和 1）的“无情翻译官”。

**在超声系统中的作用**：
探头捕获到的回波是动态范围极大的模拟电压（强反射极强，微小裂纹极弱）。ADC 负责将这些连续的起伏实时切片并量化为数字信号，供后续 FPGA 进行处理。

**核心参数与系统意义**：
* **位宽 / 精度 (Resolution / Bit-width)**：如 12-bit, 14-bit。它代表“量化尺子”的刻度密度。位宽越大，量化底噪越低，系统越能在狂风骤雨般的强回波中听清微弱的“蚊子叫”（微小缺陷信号）。
* **采样率 (Sampling Rate)**：如 50 MSPS。每秒采样的次数，根据奈奎斯特定理，必须至少是被测最高频率信号的两倍。
