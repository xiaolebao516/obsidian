# 超声探头 (Ultrasound Probe / Transducer)

#Hardware #Acoustics #传感器

**核心定义**：
超声探头是系统的“嘴巴和耳朵”，负责在物理声学世界与模拟电子世界之间进行**声电转换**。

**工作原理**：
基于**压电效应 (Piezoelectric Effect)**。施加电压时，压电晶片发生机械振动发出声波；声波反射回来撞击晶片时，晶片形变产生微弱的模拟电压。

**核心参数与系统意义**：
* **阵元数 / 通道数 (Elements / Channels)**：决定了物理孔径的大小。通道数越多，波束干涉叠加后的焦点越尖锐（横向分辨率越高），旁瓣压制越好，信噪比更高（即“看的范围更广、细节更清晰”）。
* **中心频率 (Frequency)**：决定了探测的精度与穿透力。高频看浅层细节，低频看深层穿透。

**关联概念**：
* 上游：[[物理声学]]、[[无损检测 (NDT)]]
* 下游：[[ADC]] (探头输出的微弱模拟电压直接喂给 ADC 的输入引脚)
